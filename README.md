# MCON2CODM

Makes any Bluetooth controller show up as an **Xbox Wireless Controller** on
Android, so games that only support Xbox controllers (CODM, etc.) recognize
it natively.

---

## Install

Grab the APK from the [Releases](../../releases) page and install it.

---

## First-time setup

Open the app and follow the three steps:

1. **Turn on Wireless Debugging**
   Settings → System → Developer options → Wireless debugging → enable. You can also make a split screen with pairing screen so iIP and PORT auto detect. 

2. **Tap "Pair device with pairing code"**
   A 6-digit code appears.

3. **Back in MCON2CODM, enter the code and tap Pair.**

Done. Future launches skip this.

---

## Using it

1. Turn on your Bluetooth controller.
2. Open MCON2CODM — the controller shows up as "live".
3. Tap it → tap **Spoof**.
4. Launch your game. Controller works as Xbox.

Tap **Stop** (or the notification) when you're done.
